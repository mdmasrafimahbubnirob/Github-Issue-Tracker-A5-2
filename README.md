3️⃣ What is the difference between map(), filter(), and forEach()?
-> The difference between map(), filter(), and forEach() is
A. forEach() : It goes through each item in the array and does something with it. It does not give a new array back when it finish.
B. map() : It goes through each item, changes it based on a rule you give it, and then back a brand new array with those changed items.
C. filter() : It goes through the array, tests each item against a condition, and back a new array containing only the items that passed the test.
